# Weather App

A modern Python Tkinter Weather Application developed as part of the Oasis Infobyte Python Programming Internship.

## Features
- Real-time Weather Information
- City-based Weather Search
- Temperature Display
- Humidity and Wind Speed
- Weather Emojis
- Modern Dark UI
- Error Handling

## Technologies Used
- Python
- Tkinter
- Requests API
- OpenWeather API

## How to Run

```bash
python weather_app.py
